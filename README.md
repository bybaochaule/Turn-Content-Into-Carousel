# Turn Content Into Carousel

Convert articles, newsletters, transcripts, notes, reports, threads, outlines, and rough ideas into design-tool-free social media carousel storyboards.

This Codex skill helps create source-faithful carousel packages for LinkedIn, Instagram, TikTok photo-mode, PDFs, and presentation-style posts. It focuses on message structure, swipe pacing, slide copy, visual cues, captions, calls to action, accessibility notes, and quality checks without requiring Canva, Figma, or image-generation tools.

## What It Creates

- Slide-by-slide carousel storyboards
- Hook options and carousel angles
- LinkedIn, Instagram, TikTok photo-mode, PDF, or generic carousel drafts
- Slide headlines, body copy, visual cues, and alt-text notes
- Captions, post copy, CTA options, and publishing notes
- Audits of existing carousel drafts
- Claims-to-verify and quality-control checks
